**Lark** is a simple Java framework based on Spring Boot.

## 模块说明

* lark-core: 常用组件、工具类和语言扩展
* lark-net: 网络通讯服务
* lark-pb: Protocol Buffer 序列化支持
* lark-db: 数据库访问相关组件
* lark-web: Web 相关组件
* lark-task: 分布式任务调度(基于[Skynet](https://github.com/cuigh/skynet))
* lark-msg: 消息队列
* lark-autoconfigure: 自动配置
